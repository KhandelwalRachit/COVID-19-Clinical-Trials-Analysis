# COVID-Clinical-Trials-Analysis
## 📌 Project Overview
This project analyzes global COVID-19 clinical trials data to understand research trends during the pandemic. The analysis focuses on trial status, countries conducting research, and enrollment patterns.

## 📊 Dataset
The dataset includes:
- Trial ID
- Study status
- Gender eligibility
- Enrollment numbers
- Locations
- Study phases

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 🔍 Project Workflow
1. Data Cleaning
2. Missing Value Handling
3. Feature Engineering
4. Exploratory Data Analysis
5. Visualization

## 📈 Visualizations
- Trials by country
- Trial status distribution
- Gender eligibility analysis
- Enrollment distribution
- Clinical trials over time

## 🎯 Key Insights
- The United States conducted the largest number of COVID clinical trials.
- Most trials were either recruiting or completed.
